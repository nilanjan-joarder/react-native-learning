# React Native Learning Repository

A personal repository for learning and experimenting with React Native, React fundamentals, TypeScript, and modern mobile application development.

## Purpose

This repository serves as a playground for:

* Learning React Native concepts
* Understanding React architecture and state management
* Practicing TypeScript
* Building reusable UI components
* Experimenting with mobile application features
* Exploring React Native best practices

## Topics Covered

* React Fundamentals
* React Native Components
* JSX
* TypeScript
* Props
* State Management
* Event Handling
* Conditional Rendering
* List Rendering
* Component Composition
* Styling with StyleSheet
* Navigation
* API Integration
* Mobile UI Development

## Tech Stack

* React Native
* Expo
* TypeScript
* JavaScript

## Learning Approach

This repository focuses on understanding:

* How React works
* How data flows through applications
* How to structure components
* When and where state should live
* Building scalable and maintainable interfaces

## Disclaimer

This repository is intended for learning, experimentation, and practice. Code quality, architecture, and implementation patterns may evolve as new concepts are explored and understanding improves.
